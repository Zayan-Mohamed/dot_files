# Neovim Quick Reference

## Leader Key: `,` (comma)

## Most Common Commands

### Search & Find

- `,ff` - Find files (fuzzy search)
- `,fg` - Search text in all files (live grep)
- `,fb` - List open buffers
- `,fr` - Recent files

### File Operations

- `,w` - Save file
- `,q` - Save and quit
- `<space>s` - Toggle file explorer

### Code Navigation (LSP)

- `gd` - Go to definition
- `K` - Show documentation
- `<space>rn` - Rename variable
- `<space>ca` - Code actions

### Buffer Management

- `\d` - Delete current buffer
- `gb` / `gB` - Next/Previous buffer

### Editing

- `gc` - Toggle comment (line or selection)
- `,y` - Copy entire buffer
- `,<space>` - Remove trailing whitespace

### Windows

- `←→↑↓` - Navigate between windows

## Tips

- Press `,` or `<space>` and wait to see all available keybindings
- Use `:help <topic>` for detailed help
- See full guide: ~/dot_files/NVIM_KEYBINDINGS.md
